Metashape f Calculator

焦点距離、センサーサイズ、画像サイズから、Metashape の Precalibrated f に入れるピクセル単位の値と pixel size を計算します。

元式は 360Cam-PGM-3DGS-Tools の gs360_360PerspCut.py と同じ考え方です。
pixel_size = sensor_width_mm / image_width_px
metashape_f = focal_length_mm / pixel_size

Input

Lens
Sensor mm
Image px

Output

Metashape Precalibrated f 3727.05882 Metashape の f 欄へ入れるピクセル単位の焦点距離。
pixel size 0.003756 mm 3.7563 micrometer
f_x / f_y 3727.05882 / 3727.05882 通常は f_x と f_y が一致する設定が扱いやすい。
FOV 103.72 / 80.73 deg 水平 / 垂直の画角。
既存ツールの Metashape 表示は横方向の pixel size を使っています。画像とセンサーの縦横比が一致しない場合は、入力値やクロップ後の有効センサー幅を見直してください。
